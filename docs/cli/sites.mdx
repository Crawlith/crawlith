# sites

The `sites` command provides a directory of all websites that have been crawled and are currently stored in the Crawlith database. It displays a summary of the most recent snapshot for each site, including the number of pages crawled, the last crawl date, and the overall health score.

---

## Usage

```bash
crawlith sites [options]
```

---

## Options

### `--format`

**Type:** string (`pretty` | `json`)
**Default:** `pretty`
**Required:** No

Controls the output format of the site list.

**Why It Matters for SEO**
Use `json` to programmatically retrieve a complete list of all audited sites and their current status for integration into external reporting tools or dashboards.

---

## Example

```bash
crawlith sites --format json
```

Returns a JSON array of all tracked sites, including their IDs, domains, and latest snapshot metrics.

---

## When To Adjust This

- **Centralized Reporting:** Use `--format json` to feed the list of tracked sites into a custom dashboard or monitoring system.
