# Output Files

Crawlith writes crawl outputs to your selected output folder.

## `graph.json`
Primary crawl snapshot. Contains discovered pages, internal links, and crawl metadata. Replaces the old `crawl.json`.

## `metrics.json`
Calculated graph metrics including PageRank scores, depth distribution, and structural entropy.

## `sitegraph.html` (with `--visualize`)
Enhanced interactive D3.js visualization. This is the primary visual report for exploring the link graph.

## `graph.html`
Legacy HTML report containing basic site list and metrics.

## `summary.md` (with `--markdown`)
A concise Markdown report suitable for inclusion in Git repositories or issue comments.

## `nodes.csv` and `edges.csv` (with `--csv`)
Raw data exports for use in Excel, Google Sheets, or advanced graph tools like Gephi.

## `crawlith.db`
All primary crawl data is persisted in a SQLite database located at `~/.crawlith/crawlith.db`. This database uses WAL (Write-Ahead Logging) for high-performance concurrent access.

## `crawlith-export-*.json` (generated by `crawlith export`)
Full data dump of a specific site snapshot, including all pages, edges, and calculated metrics. This is the recommended format for machine-to-machine integration.

## `diff.json`
Created when you run compare mode between two graph snapshots.
