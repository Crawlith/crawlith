# Heading Health Plugin

Analyzes heading structure, hierarchy health, and content distribution across pages.

## Features
- **Hierarchy Audit:** Detects missing H1s, multiple H1s, and hierarchy skips (e.g., H2 to H4).
- **Structural Entropy:** Measures the diversity and balance of heading levels.
- **Content Mapping:** Visualizes the document outline with parent-child relationships.
- **Divergence Check:** Flags when H1 text significantly differs from the page title.
